**HMR Badge** — uppercase pill, letter-spaced. Hero eyebrows (`brand`), live/status tags (`success`/`warning`/`danger`), neutral meta.

```jsx
<Badge tone="brand">Smart Mobile Advisor</Badge>
<Badge tone="success" dot>Knowledge base live</Badge>
```

Props: `tone`, `dot` (leading glow dot). Always short text — never a sentence.
