**HMR GlassCard** — the core glassmorphism surface for any floating panel (feature cards, stat tiles, contact cards, the chat frame).

```jsx
<GlassCard>
  <h3>Used phone guidance</h3>
  <p>Fair-price checks, inspection checklists, registry status.</p>
</GlassCard>
```

Props: `strong` (more opaque fill), `hover` (lift + cyan glow on hover, default on), `padding`. Always sits over the navy + glow + particle background.
