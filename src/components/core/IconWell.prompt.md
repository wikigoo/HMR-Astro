**HMR IconWell** — the translucent cyan-tinted tile that holds a Lucide line icon. Sits at the top of feature/pillar cards and in nav.

```jsx
<IconWell accent="cyan"><i data-lucide="shield-check"></i></IconWell>
```

Props: `size` (px), `accent` (`cyan` | `blue` | `indigo`). Intentionally semi-transparent so the background reads through.
